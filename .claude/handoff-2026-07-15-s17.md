## Gotchas hit this session

- `text=` in Playwright does NOT match input placeholders — use attribute selectors.
- React SSR splits interpolated JSX text with `<!-- -->` comments — don't substring-match across interpolations when checking rendered HTML via curl.
- `cn()` in this repo is a plain join (no tailwind-merge) — **conflicting Tailwind classes via Button `className` overrides are unreliable**; use a plain `<button>` for one-off styles (done for the red decline buttons).
- Wikimedia thumb downloads 429 without a User-Agent header; send one + small delays.
